# KAYMAX Maritime Academy Structure

This folder is the planning base for building the SMCP Trainer academy content.

The academy is divided into two schools:

1. Deck Department Academy
2. Engine Department Academy

Each school contains levels, courses, lessons, and future mission ideas. These files are not yet connected to the app UI. They are the source structure we will use to generate courses, videos, quizzes, XP rewards, and certificates.

## Next content workflow

1. Select a school: Deck or Engine.
2. Select a level.
3. Select a course.
4. Generate lessons.
5. Generate mission scripts.
6. Generate YouTube search terms.
7. Generate 10 comprehension questions.
8. Add XP, badges, and certificates.

## Mission template

Each mission should eventually include:

- Video briefing
- Character briefing
- 10 comprehension questions
- SMCP vocabulary
- Practical task
- Equipment or checklist step
- Mission complete screen
- XP reward
- Training record update
