# SMCP Trainer

Maritime English Simulator for cadets.

## Objetivo

Crear una plataforma de entrenamiento en inglés marítimo basada en escenarios SMCP para:

- Piloto Naval
- Maquinista Naval

Los cadetes comienzan como **Cadet Recruit** y suben de rango según XP, calificaciones y progreso.

## Primera versión

- Página inicial azul marítimo.
- Botones de carrera: Piloto Naval y Maquinista Naval.
- Sistema inicial de rangos.
- Misiones base por carrera.
